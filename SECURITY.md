# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.9.x   | ✅        |
| < 0.9   | ❌        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: security@thefiredev.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Suggested fix (if you have one)

We will acknowledge receipt within 48 hours and provide a resolution timeline within 7 days.

## Scope

In scope:
- Warpath API endpoints
- Authentication and authorization logic
- Data access controls
- Platform integration security

Out of scope:
- Third-party platform vulnerabilities (Yelp, Reddit, etc.)
- Rate limiting/denial of service
- Social engineering

## Disclosure Policy

We follow coordinated disclosure. We ask that you give us 90 days to resolve a vulnerability before public disclosure.
