# Warpath — Intelligence Dominance

> Geopolitical intelligence + market signals + congressional trades — correlated in real-time for asymmetric advantage.

![Screenshot placeholder](https://via.placeholder.com/1200x630/0a0a0f/dc2626?text=Warpath)

Built for Claude Mythos.

## Stack

- Next.js 14 (App Router, static export)
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS
- Deploy: Netlify

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs to `out/` — deploy to Netlify as a static site.

## Features

- Radar sweep 3D visualization
- Real-time signal correlation interface
- Congressional trade tracking
- Geopolitical risk mapping
